## 🍰 쉽고 즐거운 주문제작 케이크 업체 탐색입니당도 🍰

```
최근 3년간 주문제작 케이크에 대한 폭발적인 수요 증가와
많은 소비자들이 주문제작 케이크 업체 탐색에 불편함을 느끼고 있다는 점을 기반하여, 
주문제작 케이크 업체 탐색과정을 편리하게 만들어주는 앱 서비스를 기획했습니다.

```
